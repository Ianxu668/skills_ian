# Ian CC Plugin

个人 Claude Code 技能插件市场，收录实用的开发辅助技能。

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add ~/ian_cc_plugin
```

## 使用

安装成功后，运行 `/plugin` 命令，在市场中搜索需要的技能即可安装。

## 收录技能

| 技能 | 描述 |
|------|------|
| **mcu-debug** | MCU 项目调试助手，自动分析编译错误并提供修复方案。支持 Arduino、ESP32、STM32、SiFli (SF32) 等平台。 |
