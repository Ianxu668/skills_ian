# 常用分析维度参考

## 消费电子产品通用维度

1. **性能参数** - 处理器、内存、存储
2. **连接性** - WiFi、蓝牙、NFC
3. **续航能力** - 电池容量、使用时长
4. **屏幕显示** - 分辨率、尺寸、材质
5. **防护等级** - 防水、防尘、防摔
6. **价格区间** - 入门/中端/高端

## 家居产品通用维度

1. **材质工艺** - 材料、做工、耐用性
2. **尺寸规格** - 大小、重量、容量
3. **功能特性** - 核心功能、附加功能
4. **安全认证** - 认证标准、安全等级
5. **售后服务** - 保修、客服、配件
6. **用户评价** - 好评率、复购率

## 家电产品通用维度

1. **能效等级** - 能耗标准、节能认证
2. **噪音水平** - 运行分贝、静音模式
3. **安装要求** - 空间需求、安装复杂度
4. **智能功能** - 联网控制、自动化场景
5. **维护保养** - 清洁难度、配件更换

## 服装配饰通用维度

1. **面料材质** - 主料、辅料、舒适度
2. **工艺做工** - 缝制质量、细节处理
3. **尺码规格** - 尺码范围、版型设计
4. **品牌溢价** - 品牌知名度、设计师背书
5. **季节适用** - 四季通用/季节性

## 食品饮料通用维度

1. **成分配方** - 原料来源、添加剂情况
2. **营养价值** - 热量、蛋白质、维生素等
3. **保质期** - 保鲜时长、储存条件
4. **包装设计** - 便携性、密封性、环保性
5. **口味评价** - 口感、风味、用户评价

## 美妆护肤通用维度

1. **成分分析** - 活性成分、安全性评估
2. **功效宣称** - 保湿、抗衰、美白等
3. **适用肤质** - 干性/油性/敏感肌等
4. **品牌背景** - 研发实力、品牌历史
5. **使用体验** - 质地、气味、吸收度
