# 技术资料研究来源指南

## 学术资源

### 论文数据库
- **IEEE Xplore** - 电子工程、传感器、信号处理
- **ACM Digital Library** - 计算机科学、算法
- **ScienceDirect** - 综合科技
- **Google Scholar** - 综合学术搜索
- **CNKI** - 中文学术资源

### 专利数据库
- **USPTO** (美国专利商标局) - patents.uspto.gov
- **CNIPA** (中国知识产权局) - cnipa.gov.cn
- **Espacenet** (欧洲专利局) - worldwide.espacenet.com
- **Google Patents** - patents.google.com

## 工程资源

### 芯片/传感器厂商
| 厂商类型 | 代表厂商 | 资源类型 |
|---------|---------|---------|
| 传感器 | STMicroelectronics, Texas Instruments, Analog Devices | Datasheet, App Note |
| MCU/处理器 | ARM, NXP, Microchip | Reference Manual |
| 电源管理 | TI, Maxim, Linear Tech | Design Guide |

### 技术社区
- **Stack Exchange** (Electrical Engineering, Signal Processing)
- **EE Times** - 电子工程时报
- **Hackaday** - 硬件黑客社区
- **Instructables** - DIY项目分享

## 产品分析资源

### 拆解分析
- **iFixit** - 产品拆解指南
- **Fomalhaut Techno Solutions** - 专业拆解分析
- **TechInsights** - 半导体拆解

### 评测媒体
- **AnandTech** - 深度技术评测
- **Tom's Hardware** - 硬件评测
- **DCFever** (香港) - 数码产品评测

## 搜索策略

### 关键词组合
```
核心技术概念 + principle/theory/mechanism
核心技术概念 + design/implementation
核心技术概念 + error/analysis/compensation
核心技术概念 + patent/US Patent
```

### 中英文对照搜索
- 先搜索中文了解概念
- 用英文搜索技术细节
- 用专利号追溯原始文献
